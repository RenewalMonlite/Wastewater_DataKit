Wastewater_DataKit is a general repository for codes focused on analyzing, modeling, and post-processing wastewater network modeling. It provides tools, scripts, and data to enhance hydraulic simulations, regulatory compliance, and infrastructure planning using platforms like InfoWorks ICM.
